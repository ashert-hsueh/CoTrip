# 字体文件说明

本项目需要以下字体文件：

## 1. 华文行楷 (HuaWenXingKai)
- 文件名：HuaWenXingKai.ttf
- 用途：旅行本封面标题字体
- 要求：TrueType格式 (.ttf)

## 2. Butler
- 文件名：Butler.woff2 或 Butler.woff
- 用途：旅行本封面日期和统计信息字体
- 要求：WOFF2格式 (.woff2) 或 WOFF格式 (.woff)

## 安装方法

1. 下载所需字体文件
2. 将字体文件放入当前目录 (`src/assets/fonts/`)
3. 重启开发服务器

## 注意事项

- 确保使用合法授权的字体文件
- 若无法获取指定字体，可替换为系统默认字体或其他合适字体
- 替换字体后需更新相关CSS文件中的字体名称和路径
